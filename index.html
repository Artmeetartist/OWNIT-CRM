How to Configure Google Drive API & Fix Access Denied Errors

  1. Go to the Google Cloud Console and create a new project.
  2. Enable the Google Drive API in "APIs & Services > Library".
  3. Click on OAuth consent screen on the left menu. Since your app is in "Testing", scroll down to Test users and click + Add Users.
  4. IMPORTANT: Add your Google Email (e.g., your @gmail.com) here and click Save. (If you skip this, you will get an "Error 403: access_denied" block)
  5. Go to "Credentials" and click "Create Credentials" > API Key.
  6. Click "Create Credentials" > OAuth client ID. Select "Web application" as type.

  8. Click Save. Copy the generated Client ID and API Key below.
